# DarkCheat Menu для HornyCraft

Простое, но мощное чит-меню для Ren'Py игры *HornyCraft*, созданное для упрощения игрового процесса.

> **Примечание:** Рекомендуется сделать скриншот меню, загрузить его на сервис вроде [Imgur](https://imgur.com/) и вставить ссылку на него сюда для наглядности.
> 
> `!Пример чит-меню`

## Возможности

*   **Управление ресурсами:** Добавляйте изумруды, алмазы, золото и другие ресурсы по 100 единиц за клик.
*   **Прочность инструментов:** Мгновенно восстанавливайте прочность всех ваших мечей, кирок и топоров.
*   **Редактор инвентаря:** Получайте или убирайте ключевые предметы (например, мечи, удочку), а также изменяйте количество стакающихся предметов (доски, палки, яблоки).
*   **Отношения с персонажами:** Легко изменяйте уровень отношений с ключевыми персонажами с шагом в 1 или 10 единиц.
*   **Простой интерфейс:** Все читы сгруппированы по категориям для удобной навигации.

## Установка

1.  Скачайте файл `cheats.rpy` из этого репозитория.
2.  Откройте папку с установленной игрой `Hornycraft-0.28.1-pc`.
3.  Перейдите в подпапку `game`. Полный путь будет выглядеть примерно так: `.../Hornycraft-0.28.1-pc/game/`.
4.  Поместите скачанный файл `cheats.rpy` в эту папку (`game`).
5.  Запустите игру. Если вы все сделали правильно, чит-меню будет загружено.

## Как использовать

1.  После запуска игры в правом верхнем углу экрана появится кнопка **"ЧИТЫ"**.
2.  Нажмите на нее, чтобы открыть меню читов.
3.  Используйте кнопки для изменения нужных вам значений.
4.  Нажмите **"Вернуться в игру"**, чтобы закрыть меню и продолжить игру с примененными изменениями.
